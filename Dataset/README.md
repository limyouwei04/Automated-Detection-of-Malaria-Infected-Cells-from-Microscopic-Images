This project utilized the "Malaria Cell Images Dataset" available on Kaggle (https://www.kaggle.com/datasets/iarunava/cell-images-for-detecting-malaria), uploaded by
Iarunava. This dataset is a publicly available collection of microscopic blood cell images,
originally sourced from the official National Institutes of Health (NIH) website
(https://ceb.nlm.nih.gov/repositories/malaria-datasets/). The Kaggle version was provided to
offer a more convenient and faster download option compared to the original NIH source
